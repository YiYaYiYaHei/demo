1. outline：（轮廓）是绘制于元素周围的一条线，位于边框边缘的外围，可起到突出元素的作用；
  2. outline：轮廓线不会占据空间，也不一定是矩形；
  3. outline：outline-style是必须要写的，且只能为一个值；
  4. outline：outline-color没有时，outline-color取字体颜色；
  5. outline：不显示outline-style: none;、outline-color: transparent;或outline-width: 0;。
  1. border：简写属性在一个声明设置所有的边框属性；
  2. border：会占据空间；
  3. border：border-style是必须要写的，可以为多个值；
  4. border：border-color没有时，border-color取字体颜色；
  5. border：不显示border-style: none/hidden;

outline的应用------outline-width: 5px;

button button-solid button-dotted button-dashed button-double button-groove button-ridge button-inset button-outset

border的应用------border-width: 5px;box-sizing: border-box;

button button-solid button-dotted button-dashed button-double button-groove button-ridge button-inset button-outset button-多边框

border + outline的应用------border-width: 5px;outline-width: 5px;box-sizing: border-box;
